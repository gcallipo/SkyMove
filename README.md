# SkyMove
Right Ascension motor firmware for Skywatcher EQ5 mount and similar.

Bill Of Material:
 * 1 x Arduino board ( UNO, Nano, ... )
 * 1 x DRV8825 step motor driver module
 * 1 x NEMA 17HM15-09045 step motor
 * 3 x LED ( different color )
 * 1 x Button
 * 1 x Main on/off switch ( it must handle 2A / 12V )
 * 3 x 1000 Ohm resistors ( for led )
 * 1 x EQ5 telescope mount of course :-)

HISTORY:

First implementation, Arduino code to move telescope eq mount Right Ascension.
